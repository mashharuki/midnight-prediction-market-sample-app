# Contract module
- `pkgs/contract/src/rps.compact` is the source-of-truth Compact contract.
- `src/rps-witnesses.ts` provides witness integration; `src/index.ts` exports the package API.
- `src/test/rps-simulator.ts` wraps compact-runtime simulation; `src/test/rps.test.ts` contains contract behavior tests.
- `src/managed/rps` is generated by `compact compile src/rps.compact src/managed/rps`; never edit manually.
- Build emits declarations/JS plus generated assets into `dist`.
